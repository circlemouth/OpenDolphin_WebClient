# 37 ユーティリティ導線統合確認（RUN_ID=20260112T061215Z）

## 目的
- ドロワーと既存ショートカット導線の競合有無を確認する。
- DockedUtilityPanel の開閉で主要導線が隠れないことを確認する。
- DockedUtilityPanel の状態遷移（compact/expanded）とフォーカス移動を結合検証する。
- 診療操作/病名/処方/オーダー/文書の切替が期待通り動作することを確認する。

## 実施内容
- Charts（/charts?msw=1）にて DockedUtilityPanel を開閉し、compact/expanded それぞれで ActionBar/Timeline/ORCA サマリとの重なりを確認。
- ショートカット（Ctrl+Shift+U, Alt+P）とフォーカス遷移（開閉時）を検証。
- DockedUtilityPanel のタブ切替（診療操作/処方/オーダー/文書）と病名クイック導線を確認。

## 結果
- OK:
  - DockedUtilityPanel の開閉で ActionBar/DocumentTimeline/ORCA サマリの主要導線は隠れない。
  - Ctrl+Shift+U で DockedUtilityPanel を開き、開閉時にフォーカスが移動・復帰する。
  - 診療操作/処方/オーダー/文書の切替と病名クイック導線は動作する。
- NG:
  - Alt+P（患者検索フォーカス）で `#charts-patient-search` が見つからずフォーカス移動できない。

## 課題/補足
- ChartsPage のショートカットは `#charts-patient-search` を前提にしているが、現行 UI には該当 ID が存在しないため、Alt+P/Ctrl+F が機能しない状態。

# 25 P2 改修：履歴コピー/スタンプコピー・ペースト

RUN_ID: 20251229T215000Z

## 目的
- 旧来版の履歴コピー/スタンプコピー・ペースト相当の導線を Web に追加する。

## 対象
- OrderBundleEditPanel（オーダー束/スタンプ操作）。

## 実装観点
- 履歴コピー: 既存履歴から取り込み、編集へ反映する導線を追加。
- スタンプコピー/ペースト: 既存スタンプを複製・貼り付けできる UI を追加。
- readOnly/missingMaster/fallbackUsed 時は操作不可とし、メッセージを統一。

## 依存/前提
- 履歴/スタンプ取得 API の仕様確認（不足時は server-modernized タスク起票）。

## テスト
- コピー/ペースト反映、保存時の auditEvent、ガード条件のブロック。

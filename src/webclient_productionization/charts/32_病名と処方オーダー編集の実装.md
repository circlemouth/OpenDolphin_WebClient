## 受け入れ基準 / Done
- 右固定メニューから病名編集パネルを開き、主/疑い/開始/転帰の変更が ORCA `/orca/disease` に反映される。

- すべての編集操作が監査イベントに記録され、runId 等の観測メタが残る。
- readOnly やマスタ未同期など編集不可の状態では UI が編集ブロックと理由表示を行う。

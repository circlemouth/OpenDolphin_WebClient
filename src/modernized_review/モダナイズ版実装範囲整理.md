# モダナイズ版実装範囲整理

- RUN_ID: 20251230T181014Z
- 作成日: 2025-12-30
- 対象: `web-client/src` が呼び出す API と `server-modernized` の実装の突合（連携点の棚卸）

## 判定ラベル

- **対応済**: `web-client/src` で使用している API が `server-modernized` の JAX-RS 実装に存在する
- **未対応**: `web-client/src` で使用している API に対応する `server-modernized` 実装が見当たらない
- **対象外**: 静的ファイル / MSW などサーバ実装が前提でない

- `/api01rv2/patient/outpatient`（Patients）: `server-modernized` に該当リソースなし。

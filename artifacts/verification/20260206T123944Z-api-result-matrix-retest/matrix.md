# /orca/visits/mutation Api_Result matrix retest

This run captures HTTP status + Api_Result (+ message) for:
- direct ORCA (`/api/orca11/acceptmodv2?class=01`)
- server-modernized wrapper (`POST /orca/visits/mutation`)
- web-client via Vite proxy (`POST http://localhost:5173/orca/visits/mutation`)

All response bodies stored in this folder are redacted to avoid patient PII.

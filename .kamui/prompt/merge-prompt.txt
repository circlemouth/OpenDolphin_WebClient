以下の手順でmasterブランチへのマージを実行してください：

1. 現在の作業をコミット
   - git add . でステージング
   - git commit -m "適切なコミットメッセージ" でコミット（メッセージは作業内容に応じて記述）
- 現在のディレクトリを確認

2. リポジトリのルートディレクトリに移動


3. masterブランチにチェックアウトしてマージ
   - git checkout master
   - git merge --no-ff task/元々いたディレクトリのタスクID

実際のコマンド例：
git add . && git commit -m "適切なコミットメッセージ" && 適切なディレクトリに移動 && git checkout master && git merge --no-ff task/元々いたディレクトリのタスクID
マージする時はmasterにあるステージングか未ステージの.kamui/tasks-state.jsonのmainの方を先にコミットしてからマージ作業すること

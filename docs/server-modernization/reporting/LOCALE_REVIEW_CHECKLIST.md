# 多言語帳票レビュー チェックリスト

## 日本語テンプレート (ja-JP)
- 医療用語はカルテ画面と一致しているか。
- 全角／半角の揺れがないか。
- 患者氏名・医師氏名が敬称付きで出力されているか。

## 英語テンプレート (en-US)
- 和英翻訳の監修済み語彙を使用しているか（`glossary/en-us.csv` 参照）。
- 日付書式が `MMM dd, yyyy` で統一されているか。
- スタンプ名称はカタカナを残さず、英語説明へ置き換えているか。

## 共通事項
- アクセシビリティ: ヘッダー・本文のコントラストが 4.5:1 以上か。
- PDF 署名: 署名者情報・タイムスタンプが正しいか。
- ページ設定: 余白 15mm、A4 サイズでレイアウトが崩れていないか。

# ORCA-02 `/orca/stamp` date パラメータ追加ログ（RUN_ID=20251219T131008Z）

## 目的
- `/orca/stamp/{setCd,name}` に `date` パラメータ（クエリ）を追加し、入力セット/点数の有効期間フィルタを呼び出し元で指定できるようにする。

## 実施内容
- `OrcaResource#getStamp` に `@QueryParam("date")` を追加し、既存の第3要素（`setCd,stampName,visitDate`）よりも優先して有効日を解決するように変更。
- `resolveEffectiveDate` は既存のまま使用し、`YYYYMMDD`/ISO8601 混在入力から 8 桁を抽出する仕様を維持。

## 変更ファイル
